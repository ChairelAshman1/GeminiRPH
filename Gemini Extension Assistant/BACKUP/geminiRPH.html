Pilih Mode Sistem
Panduan Eksport RPH

1. Jana RPH lengkap dahulu (Langkah 1 & 2).

2. Pergi ke Tab Google Sheet.

3. Klik pada sel KELAS/ CLASS.

4. Klik butang "🚀 Salin RPH Lengkap".

5. Sistem akan menyalin data secara automatik mengikut urutan Tema -> Tajuk -> SK -> SP -> Objektif -> Kriteria -> Aktiviti.

Klik "Stop" jika perlu membatalkan proses.

Panduan API Key

1. Pergi ke Google AI Studio.

2. Login akaun Google.

3. Klik "Get API key".

4. Klik "Create API key".

5. Copy dan Paste di sini.
